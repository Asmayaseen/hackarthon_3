# Frontend Quickstart
1. npm i next react react-dom tailwindcss shadcn-ui react-monaco-editor recharts swr lucide-react sonner
2. npx shadcn-ui@latest init (dark theme)
3. npx shadcn-ui@latest add toast button card chart
4. dev → localhost:3000/workspace → Monaco → type code → triage POST
5. /dashboard → fetch progress → chart renders
