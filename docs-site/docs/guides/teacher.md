# Teacher Guide

Monitor your class and generate AI-powered exercises for your students.

## Getting Started

1. **Login** at [learnflow.ai/login](http://localhost:3000/login)
   - Select **Teacher** role
   - Email: `rodriguez@learnflow.ai`
   - Password: `demo123`

2. You'll land on the **Teacher Dashboard** showing:
   - Class statistics (avg mastery, total students, struggle count)
   - Struggle alerts (students who need help)
   - Full class roster with progress bars
   - AI Exercise Generator

## Struggle Alerts

The red **Struggle Alerts** section shows students who need immediate help.

Triggered when a student:
- Gets quiz score < 50%
- Fails the same exercise 3+ times
- Reports being stuck

**Actions you can take:**
- Click **Create Exercise** next to a struggling student
  - Pre-fills the exercise generator with their topic
  - Set difficulty to Easy for extra practice

## AI Exercise Generator

Use the exercise generator to create custom coding challenges:

1. **Topic**: Enter the Python topic (e.g., "list comprehensions", "for loops")
2. **Difficulty**: Easy / Medium / Hard
3. **Count**: 1, 2, 3, or 5 exercises
4. Click **Generate Exercises**

The AI generates exercises with:
- Problem statement
- Starter code template
- Expected output example

Assign generated exercises to struggling students for targeted practice.

## Class Progress

The **Class Progress** section shows all students with:
- Name and current topic
- Mastery percentage bar
- Status (On Track / Needs Help)
- Last active timestamp

Click on any student row to expand their detailed profile.
