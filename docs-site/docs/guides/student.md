# Student Guide

Welcome to LearnFlow! Here's how to make the most of your AI Python tutoring experience.

## Getting Started

1. **Login** at [learnflow.ai/login](http://localhost:3000/login)
   - Email: `maya@learnflow.ai`
   - Password: `demo123`

2. You'll land on your **Dashboard** showing:
   - Overall mastery across all topics
   - Topic progress bars
   - AI Sidekick chat button

## AI Tutor Chat

Click the **purple chat button** (bottom right) to open your AI Python Tutor.

Try asking:
- "How do for loops work in Python?"
- "What's the difference between a list and a tuple?"
- "Why is my code giving an IndexError?"

The tutor adapts to your level and provides code examples.

## Code Editor

Click **Code Editor** tab in the dashboard to:

1. **Write Python code** in the Monaco editor
2. **Run it** (green button) — executes in a sandboxed environment
3. **Get AI Review** (purple button) — AI reviews your code for:
   - Correctness
   - PEP 8 style
   - Efficiency improvements

## Quizzes

Click **Take Quiz** to test your knowledge:

- 5 random questions per quiz
- Navigate with Prev/Next or click dots
- Submit when all questions answered
- See detailed results with correct answers shown

**Mastery levels:**
| Score | Level |
|-------|-------|
| 0-40% | Beginner (Red) |
| 41-70% | Learning (Yellow) |
| 71-90% | Proficient (Green) |
| 91-100% | Mastered (Blue) |

## Struggle Detection

If the AI detects you're struggling:
- Your status shows "Needs Help" (red)
- AI Sidekick automatically opens with a helpful message
- Your teacher is notified so they can create exercises for you

Struggle triggers:
- Quiz score < 50%
- Same error 3+ times
- Stuck on an exercise > 10 minutes
